# The Seerah Project

A modern, responsive, and chronological biography website exploring the life, character, and legacy of Prophet Muhammad (PBUH).

## Project Structure

```text
seerah/
│
├── index.html

├── assets/
│   ├── images/
│   └── icons/
└── README.md
```

## Current Phase: Phase 1 (Frontend Foundation)

In this initial phase, the frontend foundation has been established:
- **Responsive Layout**: Adapts gracefully to desktop, tablet, and mobile devices.
- **Sticky Navigation**: A glassmorphic navigation bar that shrinks on scroll and spy-tracks sections to highlight current active link.
- **Mobile Menu**: Responsive hamburger drawer menu with accessible touch support.
- **Mockup Sections**: Set up with semantic HTML5 elements and elegant typography using custom CSS variables (emerald, gold, and cream).
- **History Timeline**: Chronological framework ready for content entry.
- **Media Gallery**: Placeholders for historical visual context.
- **Disabled Contact Form**: A styled frontend contact section designed for prospective inquiries.

## How to Run Locally

Since this is currently a static frontend codebase, you can run the project in any of the following ways:

1. **Directly in Browser**:
   Open [index.html](index.html) in your preferred web browser.
   
2. **Local Development Server**:
   If you have Python installed, you can launch a local server by running:
   ```bash
   python -m http.server 8000
   ```
   Then navigate to `http://localhost:8000`.
